# Guides

There comes a point in time where you just want to run something yourself and you don't want to hassle of having to find a SaaS solution with a fixed monthly fee, this is where self-hosting comes into play (it's also a great way for learning new technologies).

Some part of this category such as virtualisation and containerisation basics are found under 'the lab' documentation as they are required to build out the backbone.
