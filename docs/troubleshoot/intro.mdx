If you're having some issues with your Minecraft server then we have some things you can try, our resolution topics will cover common issues with Java and Networking as these are the most 'neutral topics' we can pick while still being helpful.

Should you need specific advice the [Admincraft Discord server](https://discord.gg/WYYxXBRUwE) has a community of over 5000 people that can assist you on a more personal level.
